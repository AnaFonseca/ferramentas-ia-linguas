# Ferramentas digitais e IA para professores/as de línguas

Página de apoio à formação de professores/as de línguas, com cartões informativos sobre ferramentas digitais e de inteligência artificial generativa.

Inclui sínteses sobre:
- o que são as ferramentas;
- como podem ser usadas por professores/formandos/as;
- como podem ser usadas por alunos/as;
- funcionalidades gratuitas e pagas;
- limitações, riscos e cuidados éticos.
